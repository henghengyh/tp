## **CampusConnect**

[![CI Status](https://github.com/se-edu/addressbook-level3/workflows/Java%20CI/badge.svg)](https://github.com/se-edu/addressbook-level3/actions)

**CampusConnect is a desktop contact management application designed for student leaders who need to coordinate across multiple university committees.**

While it provides a Graphical User Interface (GUI), it is optimised for fast Command Line Interface (CLI) interactions, allowing users to efficiently manage large volumes of contacts.

CampusConnect is especially suited for **NUSSU secretaries and student leaders** who need to:
- Organise contacts across different committees
- Quickly retrieve contact details
- Track availability and scheduling conflicts

By enabling rapid command-based workflows, CampusConnect helps users **save time, reduce coordination friction, and manage complex networks of people more effectively**.

- Tagging and filtering of contacts
- Busy period tracking and conflict detection
- Fast search across names and tags
- Sorting and bulk operations
